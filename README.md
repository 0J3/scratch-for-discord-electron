# scratch-for-discord-electron

made for a friend (normalcat)

## Setting Up

clone the dir, run `yarn`

## Running

use `yarn run pack` then run the exe in the dist\\win-unpacked folder

### NOTICE

`yarn start` will trigger an error

## Requirements

- Windows @ whatever shut up
- NodeJS @ ^12.0.0
- Yarn @ ^2.0.0
